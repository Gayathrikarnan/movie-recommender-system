# Movie-recommender-system
An AI-powered Movie Recommendation System using NLP &amp; Cosine Similarity, deployed as an interactive Streamlit web app.

# Overview

The Movie Recommendation System is a Content-Based Filtering application that suggests similar movies based on user input.

This system leverages Natural Language Processing (NLP) techniques and Cosine Similarity to compute similarity between movies based on their metadata such as genres, keywords, cast, and overview.

The application is deployed as an interactive web app using Streamlit.

# Key Features

Search any movie from the dataset 

Get top 5 similar movie recommendations

Fast similarity computation using vectorization

Interactive Web Interface with Streamlit

Machine Learning powered recommendation logic

# Methodology

The system follows a Content-Based Filtering approach:

Data preprocessing and feature engineering

Text vectorization using CountVectorizer

Similarity computation using Cosine Similarity

Ranking and returning top-N similar movies

# Tech Stack

Programming Language: Python

Data Handling: Pandas, NumPy

Machine Learning: Scikit-learn

Web Framework: Streamlit

Dataset: TMDB Movies Metadata
